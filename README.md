# BudgetBeat - Personal Expense Manager 💰

A beautiful, feature-rich personal expense tracking web application built with HTML, CSS, and vanilla JavaScript.

## Features ✨

- **Dashboard** - Financial overview with summary cards, category breakdown chart, and monthly trend chart
- **Expense Management** - Add, edit, delete expenses with full CRUD operations
- **Smart Search & Filters** - Search by description, filter by category/month, sort by various criteria
- **Beautiful Charts** - Doughnut chart for category breakdown, line chart for monthly trends, bar chart for daily spending
- **Quick Add** - One-click buttons for common expenses (Coffee, Bus, Lunch, etc.)
- **Analytics Page** - Deep insights with spending stats and visualizations
- **Dark Mode** - Toggle between light and dark themes
- **Persistent Storage** - All data saved to LocalStorage
- **Responsive Design** - Works perfectly on desktop and mobile
- **Delete Confirmation** - Modal confirmation before deleting expenses

## How to Use 🚀

1. Open `index.html` in your browser (or use a local server)
2. Add expenses via the **Add Expense** page or use **Quick Add** buttons
3. View all expenses on the **All Expenses** page with search and filters
4. See your financial overview on the **Dashboard**
5. Dive deep into **Analytics** for spending insights
